Контроллер необходим для подключения к сети одной или несколько RGB гирлянд.<br/> С помощью контроллера можно создавать неповторимые светодинамические эффекты на расстоянии (ИК-пульт ДУ в комплекте). Может использоваться как внутри помещений, так и снаружи. <br/> Контроллер подходит для следующих RGB гирлянд:<br/> • &#34;Нить&#34; 10м (арт. 245-409)<br/> • &#34;Светодиодный Дождь&#34; (Занавес) 2*1,5 м (арт. 245-309)<br/> • &#34;Светодиодный Дождь&#34; (Занавес) 2*3 м (арт. 245-319)<br/> • &#34;Светодиодный Айсикл&#34; (Бахрома) 4*0,8м (арт. 245-209)<br/> • Набор Дюралайта 6м (арт. 245-109)<br/> • Набор Дюралайта 14 м (арт. 245-119)<br/> <br/> Поддерживает 6000 LED при подключении гирлянд Дождь/Бахрома/Нить или 60м Дюралайта<br/> <br/> Контроллер для RGB гирлянд с пультом позволяет выбирать у гирлянд один из цветов свечения:<br/> -красный<br/> -зеленый<br/> -синий<br/> -желтый<br/> -голубой<br/> -пурпурный (фиолетовый)<br/> -белый <br/> -мультиколор (разноцветный)<br/> <br/> К любому из перечисленных цветов можно применить режимы:<br/> - мигание<br/> - затухание ( нарастающий и убывающий эффект )<br/> - хаотичное мерцание <br/> - мерцание белым цветом с использованием выбранного основного цвета<br/> <br/> &#34;play&#34; режим запускает цикличную смену эффекта мерцания белым цветом для всех цветов поочередно:<br/> 1. постоянное свечение мультиколор +мерцание белого цвета <br/> 2. постоянное свечение красного + мерцание белого цвета <br/> 3. постоянное свечение зеленого +мерцание белого цвета <br/> 4. постоянное свечение синего +мерцание белого цвета <br/> 5 постоянное свечение пурпурного + мерцание белого цвета <br/> 6. постоянное свечение желтого + мерцание белого цвета <br/> 7 постоянное свечение голубого + мерцание белого цвета <br/> 8. постоянное свечение белого цвета + мерцание белого цвета <br/> <br/> + и - <br/> регулировка скорости воспроизведения некоторых эффектов
